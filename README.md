Supplier Price Analysis with Python

This project is part of my journey toward becoming a Data Analyst.

Instead of focusing only on basic exercises, I started applying Python to a real-world scenario: comparing supplier prices — something directly connected to business decision-making.

Project Overview

The goal of this project is simple but practical:

Store supplier prices
Identify the cheapest and most expensive supplier
Calculate the percentage difference between them

This type of analysis is commonly used in purchasing and cost optimization.

How It Works

The script:

Stores supplier prices in a dictionary
Automatically finds:
The cheapest supplier
The most expensive supplier
Calculates the percentage difference using:

[
\text{Variation (%)} = \frac{\text{Highest Price} - \text{Lowest Price}}{\text{Lowest Price}} \times 100
]

Displays the results in a clear and readable format
